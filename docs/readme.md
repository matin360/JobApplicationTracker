# Job Application Tracker

A self-hostable job application tracker for organizing applications, follow-ups, interviews, notes, and resume versions in one place.

## Why this exists

Job searching quickly becomes messy when applications are spread across spreadsheets, email, notes, and calendar reminders. This app gives you one private place to track the full process from first application to final outcome.

## Features

- Application tracking.
- Company tracking.
- Notes and reminders.
- Interview tracking.
- Resume version tracking.
- Dashboard with status overview.
- Search, filter, and sort.
- CSV export.
- Self-hostable setup.

## Local setup

### Prerequisites
- Node.js
- PostgreSQL
- npm, pnpm, or yarn

### Setup
1. Clone the repo.
2. Install dependencies.
3. Configure environment variables.
4. Run database migrations.
5. Start the frontend and backend.
